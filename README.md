# Portfolio

Personal portfolio website showcasing projects and technical skills.

[![Live Demo](https://img.shields.io/badge/üåê_Live-Demo-2ea44f?style=for-the-badge)](https://cloud.umami.is/q/ZjTNYCHrL)

## Features

- **Modern UI** - Clean design built with React and Tailwind CSS
- **Multi-language Support** - Internationalization with i18next
- **Dynamic Project Showcase** - GitHub API integration for live repository data
- **Contact Form** - Contact form powered by [Formspark](https://formspark.io/)

## Tech Stack

- **Frontend**: React 19, Vite
- **Styling**: Tailwind CSS
- **UI Components**: Radix UI, Lucide React icons
- **Routing**: React Router
- **Internationalization**: i18next
- **API**: GitHub API
- **Form**: [Formspark](https://formspark.io/)
