# Hillnose - Personal IT Portfolio

A professional personal portfolio website showcasing my skills, technologies, and projects.

## Features

- **About Section**: Introduction and background information
- **Technologies**: Comprehensive list of technologies and tools I work with
  - Front-End: HTML5, CSS3, JavaScript, React, Vue.js
  - Back-End: Node.js, Python, Django, Express.js, FastAPI
  - Database: PostgreSQL, MongoDB, MySQL, Redis, SQLite
  - DevOps & Tools: Git, Docker, AWS, CI/CD, Linux
- **Projects**: Portfolio of completed projects with descriptions and tech stacks
- **Contact**: Email and social media links (GitHub, LinkedIn, Twitter)
- **Responsive Design**: Mobile-friendly layout

## Getting Started

To view the portfolio locally:

1. Clone the repository
2. Open `index.html` in your web browser

Or use a local server:

```bash
python3 -m http.server 8000
```

Then navigate to `http://localhost:8000/index.html`
